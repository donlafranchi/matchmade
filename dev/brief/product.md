# Matchmade Product Brief

## Vision
AI-powered matching platform connecting users based on intelligent compatibility

## Current Phase
MVP - Web Application

## Target Platforms
- **Now**: Next.js web app
- **Future**: Native iOS and Android apps

## Core Features

### MVP (In Progress)
- [ ] User onboarding
- [ ] Profile creation
- [ ] AI chat interface for profile building
- [ ] Matching algorithm
- [ ] Match display
- [ ] Basic messaging

### Post-MVP
- [ ] Advanced matching filters
- [ ] In-app notifications
- [ ] Video profiles
- [ ] Premium features
- [ ] Analytics dashboard

## Technical Stack
- **Frontend**: Next.js 14, React, TypeScript
- **Backend**: Next.js API routes
- **Database**: PostgreSQL + Prisma
- **AI**: OpenAI API (chat, embeddings)
- **Auth**: NextAuth.js
- **Hosting**: Vercel

## User Flow
1. Sign up / Login
2. Onboarding conversation with AI
3. AI builds profile from conversation
4. View matches
5. Connect with matches
6. Message matches
